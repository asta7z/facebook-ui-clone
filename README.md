# Facebook UI Clone

## Requirements:
- **Node.Js** installed **(V20.14.0 or above...)**
## Run Locally

Clone the project

```bash
  git clone https://github.com/asta7z/facebook-ui-clone
```

Go to the project directory

```bash
  cd facebook-ui-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
